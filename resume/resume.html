自学项目和作品

1.Python-Web

通过使用 Python 实现一个轻型 Web 框架，学习了其中必备模块的设计，如数据库模块、会话模块、异常处理模块、WSGI 模块、模版引擎模块、路由模块、视图模块、辅助模块，最终了解到不同于 C 语言的现代语言风格，掌握了 HTTP 基础知识、Session 的作用和路由的概念，理解了 MVC 设计模式以及数据模版分离的设计理念。

2.Travel

通过 vue-cli 脚手架搭建一个旅游应用，通过学习我理解了 Vue 生命周期的应用、组件之间的传值和事件冒泡，认识到 Vue 是数据驱动的框架。项目中还应用 Vue-Router 来完成城市选择，应用 Vuex 实现数据共享，使用 Axios 动态获取本地 mock 数据实现前后端分离，使用占位框解决轮播图加载存在的抖动，通过函数防抖控制滚动字母表的事件触发频率，利用 localStorage 存储用户数据。

3.Webpack-Learning

通过动手实践 16 个 demo 并整理笔记学习和理解了 webpack 的核心概念及经典的处理方案，如单页面代码分割懒加载、多页面打包公共代码、CSS 提取、第三方库以及图片字体处理等，理解开发环境和生产环境配置的不同。

4.Vue-SSR-Todo

深入学习 Vue 开发过程：

  1. 先在 Vue.js 框架下使用 .vue 文件开发方式完成一个代办清单应用
  2. 使用 webpack 加载静态资源及预处理器，使用 webpack-dev-server 生成 server 并启动热更替方便开发
  3. 在根组件中集成 Vue-Router 定位路由的跳转并实现懒加载
  4. 使用 Vuex 集中式地存储所有组件的数据状态(Store)并管理数据的变化(Mutation,Action)
  5. 使用服务端渲染 SSR 的方式开发 todo 应用，服务端渲染的特点在于字符串的拼接在服务端完成。后端使用 Koa 搭建 node server
  6. 设计符合 RESTful 规范的数据接口，开发时使用 Chrome 插件 Restlet Client 调试 api，正式环境 向 apiCloud 请求数据。
  7. 最后我将应用上传到阿里云服务器并使用进程管理工具 pm2 把应用部署在服务器上，用 Nginx 做反向代理把用户请求代理到 8888 端口。
